# Actix Web - Load Static Json File

## Basic Usage

```
$ cargo run
$ curl http://127.0.0.1:8080/user/3
{"id":3,"first_name":"Yuki","last_name":"Sato","username":"yuki_s","email":"yuki.sato@example.com"}
```

GET 
```
http://127.0.0.1:8080/user/{id}
```
